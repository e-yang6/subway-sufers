# Subway Surfers Body Controller

Control Subway Surfers with your body using a webcam and pose detection.

**Lean** to switch lanes, **jump** to jump, **crouch** to roll.

## Run

```
pip install -r requirements.txt
python controller.py
```

Stand still during the 3-second calibration, then play. Press `q` to quit.
